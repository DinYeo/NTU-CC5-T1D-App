# T1D Management Dashboard

## Overview
A high-end, dark-themed dashboard for Type 1 Diabetes management. This app simulates a wearable bracelet integration where users can log meals and insulin via tap gestures.

## Features
- **Bracelet Simulator**: 
    - **1 Tap**: Logs a "Meal" event.
    - **2 Taps**: Logs an "Insulin" event.
    - Visual Ripple Effect and haptic-like feedback.
- **Timeline**: Visualizes your daily events in real-time.
- **Health Alerts**: Smart detection of missed insulin timings based on your custom settings.
- **Settings**: Configure your ideal meal intervals and insulin timing windows.
- **Persistent Data**: All events and settings are saved to your browser's LocalStorage.

## How to Run
Since this is a high-performance Vanilla JS application, no installation is required!

1. Open the project folder.
2. Double-click `index.html` to open it in your default web browser.

## Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Variables, Flexbox/Grid, Glassmorphism effects.
- **JavaScript (ES6+)**: Logic, State Management, DOM Manipulation.
- **FontAwesome**: Scalable vector icons.
- **Google Fonts**: 'Outfit' typeface.
